eTB Manager Desktop application

============
REQUIREMENTS
============

Java SE 6 or higher must be installed.

Go to http://www.java.com and follow the instructions there


=========================
INSTALLATION INSTRUCTIONS
=========================

Unpack the etbmapp-<version>.zip to a folder; 


===================
RUNNING eTB MANAGER
===================

Windows Operating System

1. Go to the etbmapp-<version> folder
2. Double click on the icon etbmapp-<version>


Other Operating Systems

1. Go to the etbmapp-<version> folder
2. Execute the command 

    java -jar etbmapp-<version>.jar
